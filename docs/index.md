## Overview

**OQTOPUS Test Suite** contains external testing tools for the OQTOPUS quantum computing platform.

It provides scenario tests and endurance tests to verify the end-to-end behaviour
and long-term stability of the OQTOPUS platform.

## Contents

- **[Scenario Tests](scenario-tests/README.md)** - Scenario tests for quantum job execution using runn framework
- **[Endurance Test](endurance-test/README.md)** - Endurance tests for long-term stability of the OQTOPUS platform
